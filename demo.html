how to call msig.css

You need to add this line on the header section of main/all html files :
<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/msig-tech/msig-w3-css/msig.css">
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Nanum Gothic|Old Standard TT|Suez One|IM Fell English SC">

Buttons

Description :

  • a href : link
  • button class : create html button object with class
  • msig-button : style for basic button
  • msig-button-blue : add blue button color
  • msig-button-blue-hover : add blue mouse over button color
Actual Code : <a href="#"><button class="msig-button msig-button-blue msig-button-blue-hover">Blue Button</button></a>

Margin

Margin can be used with text and object. This is the space INSIDE the block

This is the actual margin option and description :

  • msig-margin-bottom : Adding a 32 px margin at bottom
  • msig-margin-side : Adding a 16 px margin at both side
  • msig-margin-side-small : Adding a 3 px margin at both side
  • msig-margin-end : Adding a 16 px margin at top and bottom
  • msig-margin-end-large : Adding a 32 px margin at top and bottom
  • msig-margin-end-xlarge : Adding a 48 px margin at top and bottom
  • msig-margin-up-down : Adding a 16 px margin at the top and a 36 px margin at the bottom
  • msig-margin-top-member-menu : Adding a 11 px margin at top

Padding

Padding can be used with text and object. This is the space OUTSIDE the block

This is the actual padding option and description :

  • msig-padding : Adding a 32 px padding all around
  • msig-padding-bottom : Adding a 32 px padding at bottom
  • msig-padding-down : Adding a 16 px padding at bottom
  • msig-padding-top : Adding a 16 px padding at top
  • msig-padding-top-down : Adding a 16 px padding at top and bottom
  • msig-padding-top-title : Adding a 211 px padding at top
  • msig-padding-top-member-menu : Adding a 9 px padding at the top
