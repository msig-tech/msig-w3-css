Bouton Bleu

Description : Utilisez la balise <button> avec les classes msig-button et msig-button-blue pour créer un bouton bleu. L'effet de survol donnera une couleur différente.

Bouton Or

Description : Utilisez la balise <button> avec les classes msig-button et msig-button-gold pour créer un bouton doré. L'effet de survol donnera une couleur différente.

Bouton Vert

Description : Utilisez la balise <button> avec les classes msig-button et msig-button-green pour créer un bouton vert. L'effet de survol donnera une couleur différente.

Bouton Transparent

Bouton Transparent

Description : Utilisez la balise <a> avec la classe msig-button-member-top pour créer un bouton avec un fond transparent et une couleur de texte par défaut. L'effet de survol ne change pas la couleur.
